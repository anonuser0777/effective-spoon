Що таке вірус?

БІОЛОГІЧНИЙ КОД ПОЗА ЗАКОНОМ

Вірус — це субклітинний інфекційний агент, що реплікується виключно всередині живих клітин. Він не є «живим» у класичному розумінні — це програма без власного процесора, яка захоплює чужий.

ВИЗНАЧЕННЯ_ВІРУСУ.bio
// Характеристики вірусу
структура: нуклеїнова кислота + капсид
розмір: 20–300 нанометрів
реплікація: лише всередині хост-клітини
метаболізм: відсутній (паразит)
мутація: висока швидкість змін
// Вірус = зброя, яка перепрограмовує систему
СЛАЙД 03 · АНАТОМІЯ АГРЕСОРА

Будова вірусу

КАПСИД ТА НУКЛЕЇНОВА КИСЛОТА

🧬

НУКЛЕЇНОВА КИСЛОТА

ДНК або РНК — «вихідний код» вірусу. Містить інструкції для створення нових копій. Може бути одно- або дволанцюговою.

🔷

КАПСИД

Білковий «панцир» навколо геному. Захищає генетичний матеріал та допомагає вірусу прикріплятися до клітини-мішені.

🫧

ЛІПІДНА ОБОЛОНКА

Є у деяких вірусів (наприклад, COVID-19). Двошарова мембрана з шипоподібними білками для розпізнавання рецепторів.

🔑

ПОВЕРХНЕВІ БІЛКИ

«Ключі» для входу в клітину. Кожен вірус має специфічні білки для зв'язку з конкретними рецепторами хоста.

⚗️

ВІРУСНІ ФЕРМЕНТИ

Деякі віруси несуть власні ферменти (полімерази, зворотні транскриптази) для реплікації генетичного матеріалу.

📐

СИМЕТРІЯ КАПСИДУ

Ікосаедральна (кулеподібна), спіральна або комплексна форма. Математично досконала архітектура на нанорівні.

Адсорбція

ПОШУК «ЧОРНОГО ВХОДУ» · РЕЦЕПТОРИ КЛІТИНИ

Перший крок — вірус «сканує» поверхню клітини у пошуку специфічних рецепторів. Це як підбір ключа до замка.

→

Сканування поверхні: вірус хаотично переміщується та зондує мембрану клітини-хоста.

→

Розпізнавання рецептора: поверхневі білки вірусу зв'язуються з комплементарними рецепторами клітини.

→

Фіксація: утворюється міцний зв'язок. Вірус «причалює» до поверхні клітини.

02
СЛАЙД 05 · ЕТАП 2
// ПРОРИВ ЗАХИСТУ //

Проникнення

ІН'ЄКЦІЯ КОДУ

Після адсорбції вірус впроваджує свій генетичний матеріал безпосередньо у цитоплазму клітини. Захист пробито.

→

Ендоцитоз: клітина «поглинає» вірус, загортаючи його в мембранний міхурець (ендосому).

→

Злиття мембран: оболонка вірусу зливається з клітинною мембраною, вивільняючи вміст.

→

Ін'єкція ДНК/РНК: генетичний матеріал вірусу потрапляє до цитоплазми або ядра клітини.

03
СЛАЙД 06 · ЕТАП 3
// РОЗПАКУВАННЯ //

Декапсидація

РОЗПАКУВАННЯ АРХІВУ ДАНИХ

Капсид розчиняється — клітинні ферменти руйнують білкову оболонку вірусу. Генетичний код тепер вільний і готовий до дії.

→

Дія протеаз: клітинні ферменти розщеплюють білки капсиду, звільняючи нуклеїнову кислоту.

→

Транспорт до ядра: вірусний геном транспортується до ядра клітини через ядерні пори.

→

Інтеграція: деякі віруси (ретровіруси) вбудовуються у хромосоми хоста назавжди.

04
СЛАЙД 07 · ЕТАП 4
// ЗАХОПЛЕННЯ КОНТРОЛЮ //

Реплікація

ЗАХОПЛЕННЯ «ЗАВОДУ» · СИНТЕЗ ВІРУСНИХ ЧАСТИН

Вірусний геном перехоплює «виробничі потужності» клітини — рибосоми, ферменти, енергію — і запускає масове виробництво копій.

→

Транскрипція: вірусна ДНК/РНК зчитується рибосомами клітини як «наказ» на виробництво.

→

Синтез білків: клітинні рибосоми синтезують вірусні білки капсиду та поверхневі антигени.

→

Копіювання геному: вірусна полімераза створює тисячі копій генетичного матеріалу.

05
СЛАЙД 08 · ЕТАП 5
// СКЛАДАЛЬНА ЛІНІЯ //

Збірка

КОНВЕЄР НОВИХ КОПІЙ

Усі синтезовані компоненти самостійно збираються у нові вірусні частки. Цей процес відбувається автономно без участі клітинних «менеджерів».

→

Самоскладання: білки капсиду автоматично оточують нуклеїнову кислоту завдяки хімічним зв'язкам.

→

Дозрівання: нові вірусні частки проходять конформаційні зміни, набуваючи інфекційних властивостей.

→

Масштаб: за кілька годин одна клітина продукує 10 000–100 000 нових вірусних частинок.

СЛАЙД 09 · ЕТАП 6
// КРИТИЧНА ПОМИЛКА //

Вихід

СИСТЕМА ПЕРЕВАНТАЖЕНА · РУЙНУВАННЯ КЛІТИНИ

Фінальна стадія — новостворені вірусні частки залишають клітину, щоб інфікувати нові цілі. Клітина-хост знищена.

→

Лізис: клітина «вибухає» зсередини під тиском нових вірусів — усі частинки вивільняються одночасно.

→

Брунькування: оболонкові віруси «відокремлюються» від мембрани, не руйнуючи клітину одразу.

→

Поширення: тисячі нових вірусних частинок атакують сусідні клітини — цикл повторюється по колу.
